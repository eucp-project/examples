Quickplot of a 2d cube on a map¶

This example demonstrates a contour plot of global air temperature. The plot title and the labels for the axes are automatically derived from the metadata.

In [1]:
import iris
fname = iris.sample_data_path('air_temp.pp')
temperature = iris.load_cube(fname)
In [2]:
%matplotlib inline
import matplotlib.pyplot as plt
import iris.plot as iplt
import iris.quickplot as qplt
from matplotlib import rcParams
import cartopy.crs as ccrs

rcParams['figure.figsize'] = [12, 5]

First figure: contourf with axes longitude from -180 to 180

In [3]:
qplt.contourf(temperature, 15)
plt.gca().coastlines() ;

Second figure: contourf with axes longitude from 0 to 360

Note that this one requires us to use plt.subplot to set the new projection

proj = ccrs.PlateCarree(central_longitude=-180.0)
